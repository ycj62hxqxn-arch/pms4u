1. State Authority Ownership

The most critical architectural distinction is the shift from direct state mutation to the Execution Request Pattern. The operational database (SQLite) does not own the truth of a transition.

3. Immutable Event Ledger

To guarantee production-grade auditability, every state mutation triggered by the ALCATARA Gate is recorded in an append-only JSONL ledger. This converts hidden backend logic into a visible execution philosophy.
